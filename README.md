# 🎓 Sistema Escolar

## 📌 Descrição
Sistema desenvolvido para cadastro e gerenciamento de alunos e informações escolares.

## 🚀 Tecnologias
- C#
- ASP.NET Core
- SQL Server

## 🎯 Funcionalidades
- Cadastro de alunos
- Listagem de dados
- Organização de informações
